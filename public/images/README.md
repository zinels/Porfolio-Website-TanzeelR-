# Images Directory

This directory contains images used throughout the portfolio.

## Required Images

- `driving-app.jpg` - Screenshot or mockup of the Driving Behavior Research App (1200x800 recommended)
- `researchlink.jpg` - Screenshot or mockup of ResearchLink (1200x800 recommended)
- `scholarscan.jpg` - Screenshot or mockup of ScholarScan (1200x800 recommended)
- `profile.jpg` - Profile photo (800x800 recommended, square aspect ratio)

## Image Guidelines

- Use high-quality images (at least 1200px width for project images)
- Optimize images for web (use tools like ImageOptim or Squoosh)
- Maintain consistent aspect ratios
- Use JPEG format for photos, PNG for graphics with transparency

## Generating Placeholders

You can use any image editing tool or online service to create placeholder images. The images should be replaced with actual project screenshots or mockups.
